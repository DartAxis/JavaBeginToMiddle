Цикл For


Вопросы:


1) Для чего предназначен цикл For?

Для неоднократного использования како-го либо кода с возможностью конкретного указания сколько раз выполниться код.

2) Можно ли использовать вложенные циклы For?

Можно

3) Синтаксис цикла For?

for( int i = 0; i < 10; i++){
...
какие либо действия
...
}

4) какие типы данных можно использовать в блоке инициализации?
ЛЮБЫЕ!!!!

5) Область видимости переменных цикла, можно ли использовать внешние по отношению к циклу переменные в качестве счетчика?

Можно.

6) Можно ли создать бесконечный цикл For?
Можно for( ;;){}

7) Можно ли использовать несколько переменных счетчиков в цикле?
Можно
for(int i=0, j=0; i < 10 && j < 100; i++,j=j+10){
...
}


8) Что такое unreacheable statement?
Недостижимый в рамках логики программы код. Вызывает ошибку компиляции. if() - это исключение из правил

9) Для чего служит ключевое слово break?
Для прерывания либо текущего цикла либо прерывание сцикла по метке.

10) Для чего служит ключевое слово continue?
Для перехода к следующей итерации текущего цикла либо цикла по метке.
